CHOOSING THE SOUND

Use this function to choose the sound for the current target key:

function soundFor(target){
  const pack = currentPack();
  const list = pack.sounds[target] || pack.sounds.F;
  return list[setIndex % list.length];
}

This allows each key to have several possible sounds and rotate by set number.

PRELOADING THE SOUND PACK

Browsers often block audio until the user interacts with the page. Preload the selected sound pack after the user presses Start.

const audioPools = new Map();

function allSoundsForPack(pack){
  const urls = new Set();
  Object.values(pack.sounds).flat().forEach(url => urls.add(url));
  return [...urls];
}

function preloadPack(pack){
  allSoundsForPack(pack).forEach(url => {
    if(audioPools.has(url)) return;

    const pool = [];

    for(let i = 0; i < 5; i++){
      const audio = new Audio(url);
      audio.preload = "auto";
      audio.load();
      pool.push(audio);
    }

    audioPools.set(url, pool);
  });
}

PLAYING A SOUND FOR ONE SECOND

Use this function to play each sound and stop it after one second:

let stopTimers = [];

function playOneSecond(url){
  const pool = audioPools.get(url);

  if(!pool){
    fallbackClick();
    return;
  }

  let audio = pool.find(item => item.paused || item.ended) || pool[0];

  audio.pause();

  try{
    audio.currentTime = 0;
  }catch(e){}

  const p = audio.play();

  if(p && p.catch){
    p.catch(() => fallbackClick());
  }

  const timer = setTimeout(() => {
    audio.pause();

    try{
      audio.currentTime = 0;
    }catch(e){}
  }, 1000);

  stopTimers.push(timer);
}

STOPPING AUDIO

Use this when switching sound packs or restarting:

function stopAllAudio(){
  stopTimers.forEach(clearTimeout);
  stopTimers = [];

  audioPools.forEach(pool => {
    pool.forEach(audio => {
      audio.pause();

      try{
        audio.currentTime = 0;
      }catch(e){}
    });
  });
}

FALLBACK SOUND

Include a fallback sound in case a file cannot play:

function fallbackClick(){
  const ctx = new (window.AudioContext || window.webkitAudioContext)();
  const now = ctx.currentTime;
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();

  osc.type = "triangle";
  osc.frequency.setValueAtTime(190, now);
  osc.frequency.exponentialRampToValueAtTime(80, now + .16);

  gain.gain.setValueAtTime(.001, now);
  gain.gain.exponentialRampToValueAtTime(.25, now + .015);
  gain.gain.exponentialRampToValueAtTime(.001, now + .18);

  osc.connect(gain);
  gain.connect(ctx.destination);

  osc.start(now);
  osc.stop(now + .2);
}

SWITCHING SOUND PACKS

Use this pattern:

let packId = "animals";

function currentPack(){
  return PACKS[packId];
}

function switchPack(id){
  packId = id;
  stopAllAudio();
  preloadPack(currentPack());
}

START FUNCTION

The Start button must preload the current pack:

function startGame(){
  stopAllAudio();
  preloadPack(currentPack());

  setIndex = 0;
  beatIndex = 0;
  started = true;

  renderSet();
}

CORRECT KEY PLAYBACK

When the user presses the correct key, play the mapped sound:

function handleCorrect(){
  const target = currentTarget();

  playOneSecond(soundFor(target));

  beatIndex += 1;

  if(beatIndex >= currentSet().pattern.length){
    goToNextSet();
  }else{
    updateActiveUI();
  }
}

IMPORTANT SOUND REQUIREMENTS

Use direct .mp3 URLs.
Use URL-encoded file paths.
Use a PACKS object.
Map sounds to F, G, R, V, and SPACE.
Preload sounds only after the user presses Start.
Use several Audio objects per sound file.
Play the sound from the mapped key.
Stop each sound after one second.
Stop all sounds when switching sound packs.
Include a fallback click if a file does not play.
The same sound pack structure should work for Animal Sounds, 1-Second Drums, Drum Kit, Funny Sounds, Human Voice, and Sound Art.
